Our app simulates a virtual zoo, where users can get tickets and then visit various exhibits in the zoo. We use an animal API to
get information about the animals, including the image. We store users' tickets in a MongoDB collection and ensure they have a ticket before they enter the zoo.

API: https://zoo-animal-api.herokuapp.com/
